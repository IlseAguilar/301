Anidamiento de Tags

Todos los tags de HTML empiezan con el caracter < y terminan con el caracter >. Dentro de ellos va toda la información del tag, empezando por el nombre. Por ejemplo, el tag HTML (que probablemente sea el más básico de todos)

Parrafo Normal


Los tags nos permiten organizar el contenido de una página web. Es decir, gracias a una etiqueta podremos indicar que se va a mostrar un texto, una imagen, un formulario, una tabla, un link, etc.

Negritas Al primer tag se le llama etiqueta de inicio y al segundo etiqueta de final y dentro de ambos encontramos el contenido. Se le domina elemento al conjunto formado por el tag de inicio, el contenido y el tag final
